Help

Step 1. Input from homepage

  • Cell Type: should be one keyword from a name of an organ/tissue such as pancreas, lung, gastrocnemius medialis, cardiac mesoderm, or a cell line such as GM12878, Hela, K562.
  • rsID: could be a list of reference SNP ID numbers which are separated by a comma or a space.
  • Submit: Click the "Submit" button to execute keyword search and record the SNPs to be tested.

Step 2. Prepare for SNP prediction

  • Available Tasks: the corresponding available Basenji pretrained models from DNase or histone modifications. If no tasks was found, please try to modify the input content. (check cell type and rsid)
  • Predict: Click the "Predict" button to execute profiles prediction of available tasks with the reference allele and alternative allele. The alleles are prepared based on the SNP position in the center of 131kb-long DNA sequence. The prediction process may take a few minutes considering the length of the SNP list.

Step 3. SNP prediction result

  • Box plot: The difference score of available tasks are calculated and mapped to six categories of chromatin functions: "open-chromatin", "expressed", "enhancer", "promoter", "active" and "repressed". Scores of the same category are gathered in the box plot.
  • Heatmap: Comparison of each functional difference is done by Conover's test. The result of pairwise significances are gathered in the heatmap in -log(p). The category with higher significance suggests the main function of the variant effect.
  • Menu: The leftside menu column contains a "Mutation Map" and a "Return home".
    • Mutation Map: The list of target rsIDs is shown below.
    • rsIDs: Click one of the rsID link will predict a mutation map of 40-bp flanking positions from the target SNP. Inside each position 3 reference-alternative difference scores are to be calculated. This process will take more time (about 5 min) compare with the single mutation task. The circle appears in the upper left conrner while waiting for the result.
    • Return home: Go to the homepage to start a new prediction task.

Step 4. Mutation map result

  • Mutation map: Inside each position of the map, the color ‘blue’ indicates loss of function and ‘red’ indicates gain of function comparing with the reference allele of "white".
  • Conservation score: The sequence logo is plotted with LOGOmaker by a transformation of the mutation map. The higher logo indicates the more conserved region of the reference allele.
  • Menu: The leftside menu column contains a "Return result" and a "Return home".
    • Return result: Return to the SNP prediction result page to discover more information.
    • Return home: Go to the homepage to start a new prediction task.
